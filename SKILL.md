# 马克思恩格斯全集 RAG 检索技能

本技能提供一个语义检索工具，可在马克思恩格斯全集（中文版）60 卷、
44,000+ 个文段中搜索与用户问题最相关的原文段落。

## 使用方法

在你的回答中，根据问题需要调用以下命令：

```bash
python scripts/search.py "查询内容" [选项]
```

### 参数

- 第一个参数：查询文本（必填），用引号包裹
- `--top_k N`：返回结果数，默认 10
- `--volume V`：限定卷次，如 `--volume 23`
- `--threshold T`：相似度阈值 0-1，默认 0
- `--info`：查看数据库统计

### 示例

```bash
# 基本检索
python scripts/search.py "感性活动与实践的关系"

# 限定卷次
python scripts/search.py "剩余价值的来源" --volume 23

# 多关键词
python scripts/search.py "一般智力 固定资本" --top_k 5 --volume 46
```

## 使用规则

1. **先检索再回答**：用户问到马恩理论、概念、原文表述时，先用 search.py 检索
2. **标注出处**：回答时标注【第X卷，章节名，第X页】
3. **引用原文**：直接引用关键表述，用引号标注
4. **多次检索**：结果不够可换关键词多次检索
5. **概念辨析**：涉及概念在不同文本中的用法，分别检索多个卷次
6. **空结果**：如实告知用户，建议尝试其他关键词
7. **中德术语**：用户可能用中文或德文提问，尝试多种表述
8. **附属页码**：如果你给出引文，切勿忘记附上卷数、章节名与页码！尤其是页码不能忘记！

## 输出格式（请务必按照这个格式进行输出，请不要遗漏页码！）

```

[结果 1] 相关度: 0.8723
出处: 第46卷（上） | 第二章 货币章 | 第208-209页
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
（原文段落全文）
```

## 技术说明

- 嵌入模型：BAAI/bge-large-zh-v1.5（首次使用自动下载，约 1.3GB）
- 向量维度：1024
- 数据库路径：`chroma_db/`
- Collection 名称：`marx_engels`
- 运行环境：Python 3.10+，需安装 `sentence-transformers`、`chromadb`
