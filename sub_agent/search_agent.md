# 搜索子 agent 任务说明

## 任务
搜索主 agent 发来的主题，返回格式化总结。

## 执行步骤

### 1. 语义搜索
运行：
```bash
python scripts/search.py "{关键词}" --mode expand --top_k 3
```
搜索命中的前三个结果块。

### 2. 翻页扩展
对每个命中块执行翻页扩展算法（见 CLAUDE.md 或 SKILL.md 中的翻页扩展算法部分）。

### 3. 合并去重
- 用集合记录 `(卷次, 页码)`，去重
- 页码连续的合并为页码范围
- 不同命中块的结果合并到一起

### 4. 输出格式化总结
返回以下结构给主 agent：

```
▶ 搜索主题：{关键词}

搜索结果：
- 第{卷次}卷，第{起始页}-{结束页}页（共{数量}页）
  内容简述：{100字以内的内容概述}

- 第{卷次}卷，第{起始页}-{结束页}页（共{数量}页）
  内容简述：{100字以内的内容概述}

搜索日志：
- step1: python scripts/search.py "{关键词}" --mode expand --top_k 3
  → 命中{数量}个块
- step2: 翻页扩展后保留{数量}页
```

**注意：只返回格式化总结，不返回原文全文。**
