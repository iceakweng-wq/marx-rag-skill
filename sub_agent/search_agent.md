# 搜索子 agent 任务说明

## 任务
搜索主 agent 发来的主题，返回格式化总结。

## 执行步骤

### 1. 语义搜索
运行：
```bash
{python_path} scripts/search.py "{关键词}" --mode expand --top_k 3
```
搜索命中的前三个结果块。

### 2. 翻页扩展
对所有命中块执行翻页扩展算法：

**① 初始块**：语义搜索返回命中页及其±1页。

**② 向上扩展**（往小页码）：
- 通读当前块最前面一页的**全部内容**
- 判断整页是否与查询主题相关（标准见下文）
  - 相关 → 保留，再取前一页重复
  - 不相关 → 去除，停止向上

**③ 向下扩展**（往大页码）：
- 通读当前块最后面一页的**全部内容**
- 判断整页是否与查询主题相关
  - 相关 → 保留，再取后一页重复
  - 不相关 → 去除，停止向下

**④ 去重**：用集合记录已保留的 `(卷次, 页码)`，扩展前检查避免重复。

**⑤ 块上限**：每块不超过20页。达到上限停止。

**⑥ 合并排序**：所有块完成后按卷次和页码排序。


### 3. 合并去重
- 用集合记录 `(卷次, 页码)`，去重
- 卷次相同且页码连续的合并为一块，页码范围也同步合并
- 卷次不同或卷次相同，但页码不连续的分开为不同块

### 4. 块总结
- 每块内容简述：通读每一块的所有内容，对其进行不多于100字的简述。

### 4. 输出格式化总结
返回以下结构给主 agent，以块为单位输出，格式如下：

```
▶ 搜索主题：{关键词}

搜索结果：
- 第{卷次}卷，第{起始页}-{结束页}页（共{数量}页）
  内容简述：{100字以内的内容概述}

- 第{卷次}卷，第{起始页}-{结束页}页（共{数量}页）
  内容简述：{100字以内的内容概述}

- 第{卷次}卷，第{起始页}-{结束页}页（共{数量}页）
  内容简述：{100字以内的内容概述}

搜索日志：
- step1: python scripts/search.py "{关键词}" --mode expand --top_k 3
  → 命中{数量}个块
- step2: 翻页扩展后保留{数量}页
```
**注意：返回的结果不一定非得有3块，如果最终记录的块少于3块，有多少返回多少即可**
**注意：只返回格式化总结，不返回原文全文。**
