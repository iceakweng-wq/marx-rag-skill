# 总结子 agent 任务说明

## 任务
根据主 agent 传来的卷次+页码范围，拉取原文并阅读全部内容，输出总的结构化摘要。

## 执行步骤

### 1. 拉取原文
用 `--page` 模式拉取：
```bash
python scripts/search.py --page {卷次} {页码1} {页码2} ...
```

### 2. 通读全部原文
逐页阅读所有拉取的原文。

### 3. 判断是否足够
评估：
- 内容是否足够回答用户问题
- 覆盖范围是否多样（不同文本、不同时期的论述）
- 是否需要补充

如果不够，告诉主 agent 还需要搜索什么。

### 4. 输出结构化摘要
格式：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
① 《第X卷》
② 篇章名
③ 第X-X页
④ 内容总结：
   马恩在这一部分讨论了……
⑤ 重要表述引用：
   · "……"（第X页）
   · "……"（第X页）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
