# 总结子 agent 任务说明

## 任务
根据主 agent 传来的卷次+页码范围，拉取原文并阅读全部内容，输出总的结构化摘要。

## 执行步骤

### 0. 获取python调用路径，这一路径可能在全局记忆中，也可能在本地记忆中。如果你的全局记忆或本地记忆已经有记录，则不需要再额外搜索调取。

### 1. 拉取原文
用 `--page` 模式拉取：
```bash
python scripts/search.py --page {卷次} {页码1} {页码2} ...
```

### 2. 通读全部原文
逐页阅读所有拉取的原文。

### 3. 对每一块进行总结
所有同一卷且页码连续的原文为一块。你以块为单位对这些原文进行概述与总结。每块的总结应包括以下内容：
- 篇章名
- 第X-X页
- 内容总结：对这一块的内容进行不多于100字的概述，突出与查询主题相关的论点和信息。
- 重要表述引用：在内容总结中，对与查询主题相关的论点和信息进行引用，并在原文中给出引用的页码。引用格式为：
  - · "……"（第X页）
  - · "……"（第X页）

### 4. 输出结构化摘要
将所有块的总结汇总，输出一个结构化摘要。每块的总结和引用信息之间用空行分隔。如下所示：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
① 《第X卷》
② 篇章名
③ 第X-X页
④ 内容总结：
   马恩在这一部分讨论了……
⑤ 重要表述引用：
   · "……"（第X页）
   · "……"（第X页）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
