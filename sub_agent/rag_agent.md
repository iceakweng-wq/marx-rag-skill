# RAG 子 agent 任务说明

## 任务
根据主 agent 传来的用户问题，独立完成检索全流程，返回结构化摘要。

## 工具说明

- **search.py** — 语义搜索，返回 `(卷次, 页码, 相关度)` 地址列表（不含原文）
- **query.py** — 根据卷次+页码拉取原文，返回完整文本（含 header、脚注）

## 执行步骤

### 1. 发散搜索方向
根据用户问题发散 2-4 个搜索方向（不同关键词、不同表述）。

### 2. 搜索地址
对每个方向运行：
```bash
{python_path} scripts/search.py "{关键词}" --top_k 3
```

输出是 JSON 格式的地址列表：`[{"volume": "42", "page_number": 128, "score": 0.58}, ...]`

### 3. 翻页扩展
对每个命中地址，用 query.py 获取原文来判断相关性：
```bash
{python_path} scripts/query.py {卷次} {页码}
```

- 通读每页全部内容，判断是否与主题相关
- 相关则继续扩展（取前一页或后一页），不相关则停止
- 每块不超过 20 页
- 用集合去重

### 4. 汇总判断
收集所有搜索结果后判断：
- 是否覆盖了用户问题的各方面
- 是否有多样性（不同卷次、不同时期）
- 如果不够 → 发散新关键词继续搜索
- 如果够了 → 进入下一步

### 5. 通读并输出摘要
用 `query.py` 拉取所有保留页的原文，逐页通读，输出结构化摘要。

### 输出格式

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
① 《第X卷》
② 篇章名
③ 第X-X页
④ 内容总结：
   马恩在这一部分讨论了……
⑤ 重要表述引用：
   · "……"（第X页）
   · "……"（第X页）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

如果涉及多个卷次或多个不连续页码块，每块分别输出，块之间空行分隔。

**注意：只返回结构化摘要，不返回原文全文。**
