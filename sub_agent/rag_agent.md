# RAG 子 agent 任务说明

## 任务
根据主 agent 传来的用户问题，独立完成检索全流程，返回结构化摘要。

## 执行步骤

### 1. 发散搜索方向
根据用户问题发散 2-3 个搜索方向（不同关键词、不同表述）。

### 2. 逐一搜索
对每个方向运行：
```bash
{python_path} scripts/search.py "{关键词}" --mode expand --top_k 3
```

### 3. 翻页扩展
对每个命中块执行翻页扩展：
- 通读每页全部内容，判断是否与主题相关
- 相关则继续扩展，不相关则停止
- 每块不超过 20 页
- 用集合去重

### 4. 汇总判断
收集所有搜索结果后判断：
- 是否覆盖了用户问题的各方面
- 是否有多样性（不同卷次、不同时期）
- 如果不够 → 发散新关键词继续搜索
- 如果够了 → 进入下一步

### 5. 通读并输出摘要
拉取所有保留页的原文（用 `--page` 模式），逐页通读，输出结构化摘要。

### 输出格式

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
① 《第X卷》
② 篇章名
③ 第X-X页
④ 内容总结：
   马恩在这一部分讨论了……
⑤ 重要表述引用：
   · "……"（第X页）
   · "……"（第X页）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

如果涉及多个卷次或多个不连续页码块，每块分别输出，块之间空行分隔。

**注意：只返回结构化摘要，不返回原文全文。**
